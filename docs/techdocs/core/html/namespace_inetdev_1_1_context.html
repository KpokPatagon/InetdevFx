enum Inetdev.Context.PropertyType

Defines the type of properties that can be configured for application context objects.

Enumerator
Map 

The object property is a map property of key/value entries.

Null 

The object property would be assign with a value.

Ref 

The property reference another object in the object factory.

Runtime 

The property value is calculated at runtime for the specified object.

Value 

The property specify the actual value to assign.
