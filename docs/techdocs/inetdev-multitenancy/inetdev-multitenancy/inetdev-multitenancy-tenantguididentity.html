Generate

Generates the identity by concatenating the partition key with a newly generated Guid.

Signature:
public Object Generate(model)
Returns

A newly generated Id of up to 47 characters built by concatenating the partition key with a Guid.

Remarks

This implementation extracts the partition key from the ApplicationContext which must contanins a Tenant within.

Use ApplicationContext.Current.SetTenant<Tenant>(...) to set it.
