Detailed Description

A Mvc dependency resolver that uses the application context to instantiates a controller.

To use this DependencyResolver, simply set it to the DependencyResolver class into the Global.asax initialization code as:

DependencyResolver.SetResolver(new ApplicationContextDependencyResolver());
